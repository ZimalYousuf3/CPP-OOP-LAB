# C++ Object Oriented Programming
A collection of C++ programs demonstrating core Object Oriented Programming concepts, built progressively from foundational structures to full class-based design.
---
## Topics Covered
### Structures
- Declaring and initializing structures
- Accessing and displaying structure members
- Arrays of structures
- Passing structures as function parameters
- Nested structures
### Classes & Objects
- Designing classes with private and public members
- Setter (mutator) and getter (accessor) functions
- Creating and using multiple objects
- Input validation inside classes
### Constructors & Destructors
- Default constructors
- Parameterized constructors
- Constructor overloading
- Destructors and object lifecycle
---
## Programs Included
| Program | Description |

| Date Handler | Input and display dates in multiple formats |
| Employee Manager | Store employee data using structures and classes |
